# Research & Documentation

This folder contains research notes, sources, and improvement plans for the rn-store-skills project.
Each research round adds to our understanding and feeds back into improving the skill files.

## Folder Structure

```
docs/
├── README.md                  ← you are here
├── research/                  ← research rounds, numbered sequentially
│   ├── round-01-initial.md    ← initial research and sources
│   └── round-XX-topic.md     ← subsequent rounds
├── sources/                   ← links to official docs and references
│   └── official-sources.md    ← Apple/Google official documentation links
├── changelog/                 ← what changed in each skill version
│   └── v1.0.0.md
└── ideas/                     ← future improvements and feature requests
    └── backlog.md
```

## How to Use

1. Before improving the skill, read the latest research round
2. When doing new research, create a new `round-XX-topic.md` file
3. After updating skill files, document changes in `changelog/`
4. Ideas and future improvements go in `ideas/backlog.md`
