# v4.0.0

## Added

- **16 new App Store Connect CLI skills** covering build, release, metadata, signing, TestFlight, screenshots, pricing, diagnostics, and utilities
- EAS Build / EAS Submit / Fastlane alternatives in every relevant skill
- Google Play equivalents and comparison tables where applicable
- Multi-step workflow automation recipes
- RevenueCat catalog sync reference
- Locale reference data (37 App Store locales)
- Generic SKILL.md validation for all skills (not just rn-store-compliance)

## New Skills

### Build & Distribution
- `asc-xcode-build` — build, archive, export iOS/macOS apps
- `asc-build-lifecycle` — track builds, monitor processing, cleanup
- `asc-notarization` — macOS notarization (Developer ID, Electron)
- `asc-signing-setup` — certificates, profiles, bundle IDs, capabilities

### Release & Review
- `asc-release-flow` — TestFlight + App Store release with phased rollout
- `asc-testflight-ops` — groups, testers, builds, testing notes
- `asc-submission-health` — preflight technical readiness

### Metadata & Localization
- `asc-metadata-sync` — sync and translate metadata (37 locales)
- `asc-whats-new-writer` — generate localized release notes
- `asc-subscription-localize` — subscription/IAP display names
- `asc-aso-audit` — App Store Optimization audits

### Screenshots & Marketing
- `asc-screenshots` — capture, frame, upload for both stores

### Diagnostics & Pricing
- `asc-crash-triage` — TestFlight crashes, diagnostics, Sentry/Bugsnag
- `asc-pricing` — territory pricing with PPP strategies

### Utilities
- `asc-cli-usage` — CLI reference + workflow automation
- `asc-id-resolver` — name-to-ID resolution

## Changed

- `scripts/validate.js` — supports multiple skills, validates all SKILL.md files
- `package.json` — version 4.0.0, updated description and keywords
- `README.md` — complete rewrite with skill catalog, categorized tables, attribution
