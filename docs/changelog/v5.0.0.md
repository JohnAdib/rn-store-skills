# v5.0.0

Major expansion of rejection prevention coverage — 4 new rule files, expanded subscriptions, and comprehensive legal/copyright/OTA compliance.

## Added

### New Rule Files (4)
- **`rules/copyright-media.md`** — Copyright & IP for media: unlicensed images/photos, music/audio licensing, video content, font licensing, DMCA/takedown for user-uploaded content, open source license violations (GPL in App Store), AI-generated content IP issues
- **`rules/legal.md`** — Legal compliance: GDPR (EU), CCPA/CPRA (California), DMA (EU alternative payments), COPPA (children), account deletion requirement, export compliance (encryption/ECCN), EULA/Terms of Service, age rating accuracy, gambling/loot box regulations, regional laws (China, Russia, Brazil, India)
- **`rules/ota-updates.md`** — OTA update rules: what CodePush/Expo Updates can and cannot do, hidden feature activation detection, Apple 3.3.2/2.5.2 compliance, excessive update frequency
- **`rules/app-completeness.md`** — App completeness: demo accounts for review, features behind paywall, placeholder content detection, backend availability, review notes best practices

### Expanded Subscriptions
- Auto-renewal disclosure (exact required legal text for Apple + Google)
- Subscription management deep links (Apple + Google)
- Cancellation UX requirements (no dark patterns)
- Grace period and billing retry handling (StoreKit 2 + Google Play)
- Family Sharing for subscriptions
- Offer codes and promo codes
- Price increase consent flow (StoreKit 2 Message API)
- Subscription upgrade/downgrade proration modes
- Introductory offer eligibility checking
- 3 new rejection rules: dark-pattern cancellation, missing auto-renewal disclaimer, price increase without consent

### Expanded Design Rules
- Account deletion requirement (Apple 5.1.1(v)) with detection and fix
- Mandatory login gate / guest access requirement

### Expanded React Native Patterns
- IPv6-only network compliance (Apple 2.5.6) with detection
- OTA updates section (cross-ref to rules/ota-updates.md)
- New Architecture (Fabric/TurboModules) compliance implications

### Expanded Universal Checklist
- Copyright & IP section (8 items)
- Legal compliance section (8 items)
- App completeness section (7 items)
- OTA updates section (3 items)

## Changed

- Rule count: 20+ → 30+ rejection rules with detection patterns
- Rule file count: 7 → 11
- SKILL.md routing table updated with 4 new rule categories + 5 new Quick Decision Guide entries
- Package version: 4.0.0 → 5.0.0

## Stats

Apple rejected ~1.93 million of ~7.77 million app submissions in 2024 — roughly 25% of all submissions. This release adds coverage for the most common rejection categories that were previously missing from the skill pack.
